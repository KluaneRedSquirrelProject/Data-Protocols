Kluane Red Squirrel Project Database

The Kluane Red Squirrel Project is a long-term individual-based time-series of red squirrels in the southwestern Yukon Territory of Canada. The data are organized into 20 tables within a relational mySQL database. Individual tables are not available for download here as csv files. If you are interested in accessing the data please contact Andrew McAdam at the University of Colorado. Attributes are listed for all variables included within the 20 data tables. Coding descriptions and more detailed descriptions of the variables can be found within the KRSP Database Appendix.
